## Run Locally



Clone the project

```bash
  git clone https://github.com/xdeejm97/task.git
```

Go to the project directory in cmd

```bash
  cd your/path/task
```

Build project

```bash
  mvn clean install
```

Start the server

```bash
  mvn spring-boot:run
```
